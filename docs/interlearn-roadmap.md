# interlearn Roadmap

**Version:** 0.1.0
**Last updated:** 2026-02-22

## Where We Are

interlearn indexes solution docs across the monorepo and publishes both markdown and JSON outputs for cross-repo knowledge discovery. Core indexing/search/audit paths exist and are usable.

## Roadmap — Now

- [interlearn-now-doc-baseline] **Establish canonical product artifacts** for module roadmap and vision.
- [interlearn-now-index-reliability] **Harden indexing robustness** for inconsistent frontmatter and malformed markdown.
- [interlearn-now-audit-signal] **Improve audit signal quality** so reflect coverage gaps are high-confidence.

## Roadmap — Next

- [interlearn-next-query-quality] **Improve search ranking quality** for practical debugging and implementation queries.
- [interlearn-next-taxonomy] **Refine metadata taxonomy** across category, tags, and module attribution fields.
- [interlearn-next-workflow-integration] **Integrate stronger with review and planning workflows** so indexed knowledge is surfaced proactively.

## Roadmap — Later

- [interlearn-later-feedback-loop] **Add feedback loops for low-value hits** to iteratively improve ranking behavior.
- [interlearn-later-impact-metrics] **Track reuse impact metrics** across modules and sprint cycles.
- [interlearn-later-context-packaging] **Package compact context bundles** for downstream agents and tooling.

## Research Agenda

- Evaluate ranking heuristics against real incident-resolution tasks.
- Identify lightweight scoring signals for “reused successfully” outcomes.
- Define a stable schema for cross-tool consumption of index outputs.
